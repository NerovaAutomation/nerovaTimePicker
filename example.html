Implementation

Step 1: Add the script to your HTML:

<script src="https://cdn.jsdelivr.net/gh/yourusername/timepicker@main/timepicker.js"></script>

Step 2: Add data attributes to inputs:

<input data-time-picker data-minute-interval="5" placeholder="Select time">

Step 3 (Optional): Customize with CSS variables:

:root {
    --timepicker-primary: #your-color;
    --timepicker-bg: #your-bg-color;
}
